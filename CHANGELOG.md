# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-24

### Added
- Initial release of PC814 zero-crossing detection library
- Timer Input Capture support for zero-crossing detection
- Configurable pull-up/pull-down GPIO settings
- Automatic line frequency measurement (50/60 Hz)
- Period calculation between zero-crossings
- Phase angle calculations
- Zero-crossing count tracking
- Statistics collection (min, max, average)
- Callback functions for zero-crossing events
- Blocking and non-blocking wait functions
- Helper functions for common phase calculations (90°, 180°)
- **Three-phase system support**
- **Automatic phase sequence detection (ABC/ACB)**
- **Phase swap recommendations**
- **Phase imbalance detection**
- **Synchronization verification**
- Complete documentation in English
- Comprehensive usage examples

### Features
- 33+ functions for complete zero-crossing detection
- Support for single-phase and three-phase AC systems
- Production-ready for AC 220V applications
- Modular architecture for easy portability
